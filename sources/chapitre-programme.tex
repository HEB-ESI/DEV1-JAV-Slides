\end{itemize}

\begin{frame}{Un programme}
La seule chose dont est capable un ordinateur est de réaliser extrêmement rapidement des instructions élémentaires
\par\bigskip
Toute tâche qu'on veut lui confier doit donc être préalablement décrite comme une suite séquentielle d'instructions (un programme)
\note[item]{\url{http://en.wikipedia.org/wiki/Source_lines_of_code} donne des exemples de taille de programmes}
\end{frame}



\begin{frame}{Un langage}
\emph{Une classe de langages est adaptée à une classe de problèmes \dots\ et ces problèmes évoluent dans le temps \dots}
\note[item]{On peut aborder ici l'historique des langages : machine, assembleur, haut niveau, structuré, orienté objet, fonctionnels, orientés aspects\dots}
\end{frame}
